# Delivery Shortest Path – Dijkstra Algorithm

تاسك تحليل خوارزميات -- أحمد عرفات محمد محمد -- سكشن 6 -- م/ندي السيد .  
إيجاد أسرع طريق توصيل أوردر في شوارع القاهرة باستخدام خوارزمية Dijkstra

## وصف المشكلة
لما العميل يطلب أكل من المطعم، السؤال الأهم:  
السواق هيعدي منين عشان يوصل في أقل وقت وسط زحمة القاهرة دي كلها؟

بنمثل المدينة على شكل Graph:
- كل منطقة → Node  
- كل طريق بين منطقتين → Edge  
- وزن الحافة → الوقت المتوقع بالدقايق (مع الزحمة الصبح)

المطلوب: أقصر وقت من المطعم (Restaurant) للعميل (Customer) + المسار الكامل بالترتيب

## ليه اخترت Dijkstra بالذات؟
- الأوزان كلها موجبة (الوقت ما ينفعش يبقى سالب)  
- بتديني النتيجة الصحيحة 100%  
- أسرع بكتير من Bellman-Ford  
- هي نفس الخوارزمية اللي بتستخدمها تطبيقات الدليفري والخرائط

## تعقيد الخوارزمية
O((V + E) log V) باستخدام Binary Heap  
يعني شغال تمام حتى لو الخريطة كبيرة
